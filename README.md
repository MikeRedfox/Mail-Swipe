## Git Installation
```
# clone the repo
$ git clone https://github.com/MikeRedfox/Mail-Swipe

# change the working directory to Mail-Swipe
$ cd Mail-Swipe

# install the requirements
$ pip install -r requirements.txt
```

## Usage

```
python mailSwipe.py
```
